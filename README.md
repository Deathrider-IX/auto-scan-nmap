## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Deathrider-IX/auto-scan-nmap.git

# Navigate to directory
cd auto-scan-nmap

# Run the script
./scan.sh
```

---

## 📋 Requirements

- Bash shell
- Nmap installed (`apt-get install nmap` or `brew install nmap`)
- Root/sudo privileges for network scanning
